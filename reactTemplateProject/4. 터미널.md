> 터미널
---


- 터미널 기능을 구현하려면 전용 터미널 백앤드가 필요한가보다..
- 이걸 API 백엔드와 같이구성을 하기엔 부적절하다고 한다.

- frontend
    - xterm.js
        * VS Code처럼 진짜 터미널 UI 제공
        * 그냥 이걸 쓰랜다..
        * 설치
        ```
        npm install @xterm/xterm @xterm/addon-fit @xterm/addon-web-links
        ```
        * 여기선 vite.config에 설정했는데 운영시엔 nginx에서 할듯
- backend
    - Node.js + Express + node-pty
        * 아마 이쪽으로 해야하지 않을까...
        ```
        npm install express ws node-pty
        ```
        * 어차피 nodejs라 그냥 실행하면 될거같음
        * 실행
        ```
        node terminalServer.js
        ```
    - Wetty
        * 설치 후 실행
        ```
        npm install -g wetty
        wetty --port 8080
        ```
