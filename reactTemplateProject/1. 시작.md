> 시작
---
- 목적
    - Vite - react -typescript로 제작
    - 필요할때마다 생각나는거 만들기
    - 필요할 때 재활용



- create project
```sh
npm create vite@latest . -- --template react-swc-ts
```


- 초기 구성
    * 폴더 구조   
    ```
    src/
    ├── components/
    │   ├── common/      ✅ Button, Loading 컴포넌트
    │   ├── layout/      ✅ 준비됨
    │   └── ui/          ✅ 준비됨
    ├── hooks/           ✅ 6개 커스텀 훅
    ├── utils/           ✅ 4개 유틸리티 파일
    ├── types/           ✅ 공통 타입 정의
    ├── constants/       ✅ 전역 상수
    ├── pages/           ✅ 준비됨
    ├── layouts/         ✅ 준비됨
    ├── contexts/        ✅ 준비됨
    ├── services/        ✅ 준비됨
    └── styles/          ✅ 전역 스타일, CSS 변수
    ```
    * 설정
        - Vite 설정 파일 업데이트 (vite.config.ts)
        - 포트: 3000으로 설정
        - 브라우저 자동 열기
        - 경로 별칭 설정
        - 빌드 최적화
        - 환경 변수 파일 생성
        - .env (개발용)
        - .env.example (템플릿)
        - .env.production (프로덕션용)
